# AI Training & Machine Learning Policy

This document outlines the specific restrictions and requirements for using the Jarvis repository in relation to Artificial Intelligence (AI) and Machine Learning (ML) activities.

## 1. General Prohibition

Unless explicitly authorized in writing by Darsh Shah (the Licensor), the use of any portion of this repository (source code, documentation, assets, or outputs) for the purpose of training, fine-tuning, or developing AI/ML systems is strictly prohibited.

## 2. Prohibited Activities

Prohibited activities include, but are not limited to:

- **Foundation Model Training:** Using the source code as part of a dataset for pre-training foundation models (e.g., LLMs, Vision models).
- **Fine-Tuning:** Using the repository's logic or structured data to fine-tune existing models for commercial or proprietary use.
- **Embedding Generation:** Generating vector embeddings of the codebase for use in Retrieval-Augmented Generation (RAG) systems or search indexes without authorization.
- **Dataset Generation:** Using the repository to create synthetic datasets, instruction-tuning sets, or code-completion datasets.
- **Classifier Development:** Training classifiers or heuristic models to replicate the functionality or decision-making processes of the software.
- **Distillation:** Attempting to "distill" the intelligence or logic of this software into smaller or alternative models.

## 3. Commercial Inference

Building or hosting commercial inference services (e.g., APIs, web interfaces) that leverage models trained or fine-tuned on this software is prohibited without a separate commercial agreement.

## 4. Exceptions

- **Personal Research:** Use for non-commercial, personal research or educational experiments is permitted, provided that any resulting models are not deployed for profit or shared as proprietary assets.
- **Public Open-Source Training:** If a model is being trained for purely non-commercial, public open-source research (e.g., by a university or non-profit), please contact the Licensor for an explicit waiver.
