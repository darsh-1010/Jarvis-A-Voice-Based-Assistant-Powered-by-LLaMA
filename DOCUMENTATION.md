# 🧠 Jarvis AI Documentation (v2.0 - Multi-Model HUD)

Welcome to the documentation for **Jarvis v2**. This version introduces a high-end web interface, multi-model cloud intelligence, and a distributed API-first architecture.

---

## 🚀 The v2 Architecture

Jarvis now operates as a **Distributed System**:

1.  **🧠 Multi-Model Brain**:
    - **Ollama (Local)**: Private and offline.
    - **Gemini 2.0 Flash (Cloud)**: High-speed, high-reasoning, and vision-ready.
    - **OpenRouter (Cloud)**: Access to the best free open-source models.
    - Jarvis automatically switches or allows you to pick the brain that suits your needs.
1. Install dependencies: `pip install -r requirements.txt`
2. Run the server: `uvicorn app.main:app --reload`

### Frontend (Vercel Ready)
1. Navigate to `frontend/`.
2. Install: `npm install`
3. Run: `npm run dev`
4. Deploy to Vercel for the ultimate HUD experience!
